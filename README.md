## Weather Dashboard App

A modern and responsive **Weather Dashboard** built with **React (Vite + TypeScript)** and **Material UI**.  
It supports bilingual functionality (**English / Persian**) and fetches real-time weather data using the **OpenWeather API**.

---

## Features

- **Login / Logout** flow  
  Simple login screen that takes your name and navigates to the main dashboard.

- **Weather Data Integration**  
  Fetches current and future weather data using **Axios** and **OpenWeather API**.

- **Bilingual Support (EN / FA)**  
  Implemented using **i18next**, allowing users to switch between English and Persian.

- **Light / Dark Mode**  
  Toggle between themes using Material UIâ€™s theme system.

- **Fully Responsive Design**  
  Works seamlessly on desktop, tablet, and mobile devices.

- **Built with Vite + TypeScript**  
  Fast development, type safety, and optimized build performance.
